# cryptography-asn.1
Implementation of ASN.1 DER encoder that can encode subset of ASN.1 types by implementing these functions,  asn1_boolean(bool), asn1_integer(i), asn1_bitstring(bitstr), asn1_octetstring(octets), asn1_null(), asn1_objectidentifier(oid), asn1_sequence(der), asn1_set(der), asn1_printablestring(string), asn1_utctime(time), asn1_tag_explicit(der, tag), asn1_len(content)
